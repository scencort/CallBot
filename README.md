# CallBot

Telegram-бот для ведения звонка по скрипту продажи квартиры.

Что умеет:

- запускает сценарий командой `/start`
- показывает вопросы по порядку до 5 пункта
- после каждого вопроса принимает заметку одним сообщением
- показывает прогресс по шагам
- умеет копировать текущую реплику или вопрос одной кнопкой
- использует новые styled-кнопки Telegram
- в конце предлагает кнопки `Согласен` и `Не согласен`
- если клиент не согласен, бот запрашивает причину отказа
- командой `/summary` показывает текущую сводку

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

Если бот уже был установлен раньше, лучше обновить библиотеку:

```bash
pip install --upgrade -r requirements.txt
```

2. Задать токен бота в переменной окружения `TELEGRAM_BOT_TOKEN`.

3. Запустить:

```bash
python bot.py
```

## Команды

- `/start` - начать новый звонок
- `/newcall` - сбросить и начать заново
- `/summary` - показать текущие заметки
- `/cancel` - отменить текущий звонок
